강좌안내
- 여러개의 강좌를 계속 올릴수 있게 하라. 
- 왼쪽부터 오른쪽으로  계속 넘기면서 제시하라. 
- 일단은 4개 강좌를 올린다. 

- 깅좌1 AI 상세페이지
AI,인공지능 추천에 최적화된 상세페이지
상세페이지 기획 + 구성 + 디자인  
고객의 구매 패턴을 분석하여 개인 맞춤형 상품을 구성합니다. 
AI 추천 알고리즘을 최적화하여, 전환율을 높일수 있는 상세페이지를 제작합니다. 
기간. 난이도에 따라, 1주일~2주일
가격
1.	디자인 작업 : 50~80만
2.	디자인 + 세부기획 : 100~200만
3.	디자인 + 세부기획 + 영상 + 효과 : 200~500만
촬영이 필요할 경우,  추가 기간, 비용, 발생


- 강좌2 AI 쇼츠, 릴스 (30~60초)
AI,인공지능 추천에 쇼츠,릴스 기획, 제작
고객의 제품과 브랜드 특징을 고려하여, 주목도 높은 쇼츠, 릴스를 제작합니다. 
AI 추천 알고리즘을 최적화하여, 주목도, 전환율을 높일수 있는 쇼츠,릴스입니다. 
기간. 난이도에 따라, 1주일~2주일
가격
1.	단순 편집 : 30~50만
2.	기획 + 디자인 + 편집  : 50~100만
3.	기획 + 디자인 + 영상 효과 : 100~200만
4.	촬영이 필요할 경우,  추가 기간, 비용, 발생



- 강좌3. 웹페이지, 랜딩페이지 
AI, 인공지능 최적화 웹페이지 기획, 디자인, 배포
AI 기반 전략으로 웹페이지 성과를 극대화하세요.
기획부터 디자인, 배포까지 전 과정 원스톱 지원! 방문자 행동을 분석해 전환율을 높입니다. 당신의 웹페이지,랜딩페이지를  이제 인공지능에 최적화 하십시오. 
기간. 난이도에 따라, 1개월~3개월
가격
1.	브랜드, 제품, 예약, 구매, 챗봇 : 500만
2.	브랜드, 제품, 예약, 구매, 챗봇 + 로그인인증 : 700만
3.	브랜드, 제품, 예약, 구매, 챗봇 + 로그인인증 + DB  + 고객관리 : 1000만


- 강좌4. 브랜드 Naming Package 
- 현존 최고수준의 브랜드 Name Package
창업자의 철학을 담아 **직관적이고 감각적인 네이밍**을 만들어드립니다.  
스시도쿠 겐, 견체공학, 헤겔의 빛, 신성의 숲, 쁘띠쿠아, 홍국시, 도찜계찜, age button등 듣기만 하면 잊을수 없는 브랜드 이름을 지어드립니다
- 브랜드 이름, 슬로건, 약속, 페르소나등 브랜드전략의 기초를 만들어드립니다. 
1.	브랜드 이름 대안 3가지  제시
2.	고객 페르소나 정의  
3.	슬로건, 핵심 메시지, 브랜드 스토리 개발  
기간. 약 1개월 소요
가격: 500만원 (중간계 기존 수강생은 30% 할인)

제품1~제품4까지의 설명은 간단하게 보이게 하고, '더보기' 버튼을 만들고, 이 버튼을 누르면 전체보기를 하라. 



사용자후기

- 3X2 박스형태, 별 5개 후기를 AI가, 임의로 만듭니다. 
- 고객후기는 가운데  정열로하고, 줄바꿈이 보기에 좋게 만듭니다.
