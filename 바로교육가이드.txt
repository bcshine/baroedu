=====================================

📅 최종 업데이트: 2024년 12월
💻 프로젝트 상태: 2단계 인증 시스템 완료 (100%) → 3단계 사용자 기능 시작

## 🏗️ 전체 아키텍처
- **플랫폼 유형**: 온라인 VOD 기반 동영상 스트리밍 교육 서비스
- **디자인**: 모바일 반응형 웹사이트
- **폰트**: Noto Sans KR (Google Fonts)
- **색상 테마**: 메인 블루(#2c5aa0), 포인트 오렌지(#ff6b35)
- **백엔드**: Supabase (데이터베이스 + 인증)

#### 🗄️ 데이터베이스 구조 (Supabase)

**users 테이블 (Supabase auth.users 확장)**
```sql
- id (uuid, primary key)
- email (text, unique)
- name (text)
- phone (text)
- profile_image (text, URL)
- created_at (timestamp)
- updated_at (timestamp)
```

**enrollments 테이블** (수강 신청)
```sql
- id (uuid, primary key)
- user_id (uuid, foreign key)
- course_id (text)
- enrolled_at (timestamp)
- progress (jsonb) -- 각 레슨별 진행률
- completed_at (timestamp, nullable)
```

**reviews 테이블** (강의 리뷰)
```sql
- id (uuid, primary key)
- user_id (uuid, foreign key)
- course_id (text)
- rating (integer, 1-5)
- content (text)
- created_at (timestamp)
```
